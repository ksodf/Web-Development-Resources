## HTML

### References
- [MDN Web Docs – HTML](https://developer.mozilla.org/en-US/docs/Web/HTML) – Comprehensive HTML reference and tutorials by Mozilla.
- [HTML Living Standard](https://html.spec.whatwg.org/) – The official HTML specification maintained by WHATWG.
- [W3Schools HTML Tutorial](https://www.w3schools.com/html/) – Beginner-friendly HTML tutorials and references.

### Validators & Tools
- [W3C Markup Validation Service](https://validator.w3.org/) – Check your HTML markup for errors and warnings.
- [HTMLHint](https://htmlhint.com/) – Static code analysis tool for HTML files.
- [Emmet](https://emmet.io/) – Plugin for many editors that dramatically improves HTML workflow.

## CSS

### References
- [MDN Web Docs – CSS](https://developer.mozilla.org/en-US/docs/Web/CSS) – Comprehensive CSS reference and tutorials by Mozilla.
- [CSS-Tricks](https://css-tricks.com/) – Tips, tricks, and techniques on using CSS.
- [Can I Use](https://caniuse.com/) – Browser support tables for modern CSS and HTML features.

### Frameworks & Preprocessors
- [Tailwind CSS](https://tailwindcss.com/) – A utility-first CSS framework for rapid UI development.
- [Bootstrap](https://getbootstrap.com/) – Popular responsive CSS framework.
- [Sass](https://sass-lang.com/) – A powerful CSS preprocessor adding variables, nesting, and more.
- [PostCSS](https://postcss.org/) – A tool for transforming CSS with JavaScript plugins.

### Tools
- [Flexbox Froggy](https://flexboxfroggy.com/) – A fun game for learning CSS Flexbox.
- [Grid Garden](https://cssgridgarden.com/) – A fun game for learning CSS Grid.
- [CSS Gradient](https://cssgradient.io/) – Visual tool for creating CSS gradients.
- [Clippy](https://bennettfeely.com/clippy/) – CSS `clip-path` maker with visual editor.
- [Animate.css](https://animate.style/) – Ready-to-use, cross-browser CSS animations.

## JavaScript

### References
- [MDN Web Docs – JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript) – Comprehensive JS reference and tutorials.
- [JavaScript.info](https://javascript.info/) – Modern JavaScript tutorial from the basics to advanced topics.
- [ECMAScript Specification](https://tc39.es/ecma262/) – The official ECMAScript language specification.

### Libraries & Utilities
- [Lodash](https://lodash.com/) – A modern JavaScript utility library delivering modularity and performance.
- [date-fns](https://date-fns.org/) – Modern JavaScript date utility library.
- [Axios](https://axios-http.com/) – Promise-based HTTP client for the browser and Node.js.
- [Zod](https://zod.dev/) – TypeScript-first schema declaration and validation library.

### Frameworks
- [React](https://react.dev/) – A JavaScript library for building user interfaces.
- [Vue.js](https://vuejs.org/) – The progressive JavaScript framework.
- [Svelte](https://svelte.dev/) – A compiler-based UI framework that writes less code.
- [Angular](https://angular.io/) – Platform for building mobile and desktop web apps.

## Frameworks & Libraries

### Full-Stack
- [Next.js](https://nextjs.org/) – The React framework for production.
- [Nuxt.js](https://nuxtjs.org/) – The Intuitive Vue Framework.
- [Remix](https://remix.run/) – Full stack web framework focused on web standards.
- [Astro](https://astro.build/) – The all-in-one web framework for content-focused sites.

### Back-End
- [Node.js](https://nodejs.org/) – JavaScript runtime built on Chrome's V8 engine.
- [Express](https://expressjs.com/) – Fast, minimalist web framework for Node.js.
- [Fastify](https://fastify.dev/) – Fast and low overhead web framework for Node.js.

## Build Tools

- [Vite](https://vitejs.dev/) – Next generation frontend tooling.
- [Webpack](https://webpack.js.org/) – Static module bundler for modern JavaScript applications.
- [esbuild](https://esbuild.github.io/) – An extremely fast JavaScript bundler and minifier.
- [Parcel](https://parceljs.org/) – Zero configuration web application bundler.
- [Rollup](https://rollupjs.org/) – Module bundler for JavaScript.
- [Turbopack](https://turbo.build/pack) – Incremental bundler and build system optimized for JavaScript.

## Version Control

- [Git](https://git-scm.com/) – Distributed version control system.
- [GitHub](https://github.com/) – Hosting for Git repositories plus collaboration features.
- [GitLab](https://gitlab.com/) – DevOps platform with Git repository management.
- [Conventional Commits](https://www.conventionalcommits.org/) – A specification for adding human and machine readable meaning to commit messages.

## Design & UI

### Icons & Illustrations
- [Heroicons](https://heroicons.com/) – Beautiful hand-crafted SVG icons by the makers of Tailwind CSS.
- [Feather Icons](https://feathericons.com/) – Simply beautiful open-source icons.
- [Undraw](https://undraw.co/) – Open-source illustrations for any idea.

### Colors & Typography
- [Coolors](https://coolors.co/) – The super fast color palette generator.
- [Google Fonts](https://fonts.google.com/) – Free, open-source fonts optimized for the web.
- [Type Scale](https://typescale.com/) – A visual calculator for typographic scales.

### Prototyping & Wireframing
- [Figma](https://www.figma.com/) – Collaborative interface design tool.
- [Excalidraw](https://excalidraw.com/) – Virtual whiteboard for sketching hand-drawn like diagrams.

## Performance

- [Lighthouse](https://developer.chrome.com/docs/lighthouse/) – Automated tool for improving the quality of web pages.
- [WebPageTest](https://www.webpagetest.org/) – Free website performance testing tool.
- [PageSpeed Insights](https://pagespeed.web.dev/) – Reports on the performance of a page on mobile and desktop.
- [Bundlephobia](https://bundlephobia.com/) – Find the cost of adding an npm package to your bundle.

## Accessibility

- [axe DevTools](https://www.deque.com/axe/devtools/) – Browser extension for automated accessibility testing.
- [WAVE](https://wave.webaim.org/) – Web Accessibility Evaluation Tool.
- [WebAIM Contrast Checker](https://webaim.org/resources/contrastchecker/) – Check color contrast ratios for accessibility.
- [ARIA Authoring Practices Guide](https://www.w3.org/WAI/ARIA/apg/) – How to use WAI-ARIA to create accessible rich web apps.
